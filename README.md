# vue-admin


## 项目技术栈

> vue-cli3、vue-echarts、vue-router、element-ui、echarts-gl、photo-sphere-viewer（这个看package.json就可以了）

> 项目演示
![image] (https://github.com/Aliceco/vueAdmin/tree/master/src/assets/img/gtAWVDyZts.gif)

## 安装
```
yarn install
```

### 运行
```
yarn run serve
```

### 打包
```
yarn run build
```
